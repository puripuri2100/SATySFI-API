色作成関数

val gray : float -> color

グレイスケールを指定することで色を作成できます。gray 0.で黒、gray 1.で白です。

val rgb : float -> float -> float -> color

RGBでの色空間を指定することで、色を作成できます。 rgb 1. 0. 0.で赤、rgb 0. 1. 0.で緑、rgb 0. 0. 1.で青です。

module Color2: sig end

satysfi-lib /color2.satyhによって提供されます。

val srgb24 : int -> int -> int -> color

RGB値の指定に、0から255までの整数を用いる関数です。

val of-css : string -> color

#F5F5DCやbeigeを指定すると、その色がかえってきます。 登録されている色はCSSで使えるものだけです。
